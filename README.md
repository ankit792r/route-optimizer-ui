# Route Optimization Frontend

Frontend for the trucking route optimization and ELD logging system.

Built with:

- React
- Vite
- TypeScript
- TailwindCSS
- shadcn/ui
- TanStack Query
- TanStack Router
- React Leaflet

---

# Requirements

Install:

- Bun
- Node.js (optional but recommended)

Install Bun:

https://bun.sh

---

# Environment Variables

Create a `.env` file in the project root.

Example:

```env
VITE_API_BASE_URL=http://localhost:8000/api
```

# Install Dependencies

Using Bun:

```bash
bun install
```

# Run Development Server

```bash
bun run dev --port=3000
```
